# 简介

这是一个 TCP 的内网穿透项目

可以经过一个外网服务器把内网端口映射到外网，这样就可以在外网访问内网的资源

原理如下：

```
--------------                    -----------------------              --------------------
|                 |-------------> |                              |-------->  |                         |
| 内网代理 |                     |外网中继服务器    |               | 访问设备         |
|                 |<------------- |                              |<--------  |                         |
--------------                    -----------------------               -------------------

```

## 主要特性

1、支持snnapy 高速压缩
2、流量统计
3、异步IO
4、安全接入
